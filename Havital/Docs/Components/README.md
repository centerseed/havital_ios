## 概述
本資料夾包含 Havital 應用程式中所有主要組件的詳細文檔。每個組件的文檔都包含其功能說明、技術實現、使用示例等信息。

## 文檔結構
每個組件文檔都包含以下部分：
- 概述
- 主要功能
- 技術實現
- 使用示例
- 注意事項
- 相關組件
- 更新歷史

## 如何閱讀
1. 從 README.md 開始，了解整體結構
2. 根據需要查看特定組件的文檔
3. 參考相關組件文檔以了解組件間的關係

## 維護指南
1. 新增組件時，請創建對應的 .md 文件
2. 更新組件時，請同步更新文檔
3. 保持文檔結構的一致性
4. 及時更新目錄
